Taxwise 2008 for WA District 1 Instructions

Simply click the download link below and then click RUN when it asks if you want to run or save the file. If you do not get the option to RUN then save the file and double click it to run it after the download.

  1. There are periods of time when the script appears to be doing nothing. Be patient
  2. Setup will take several minutes on newer, faster computers and much, much longer on older machines, maybe even 10 minutes. Be patient if it seems to be taking a long time to update reporting capabilities and Net framework.
  3. It will put a TW2008 icon on your desktop.
  4. There will be three users: Admin, Training and MYSITE. Use MYSITE – The passwords are the same as the user ID's except for Admin.
  5. You can prepare returns just as you would on our production systems, you can also run diagnostics. However you will not be able to click efile to generate an efile on this installation. That can only be done on our production systems

This download includes the 23.15 end of season release of Taxwise and the Oregon 3 release for Oregon Taxes
